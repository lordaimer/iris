VERIFICATION

This package downloads the official Iris binary from GitHub releases.

The binary is downloaded from:
https://github.com/lordaimer/iris/releases/latest/download/iris-windows-amd64.exe

To verify the package contents:

1. Visit the GitHub releases page: https://github.com/lordaimer/iris/releases/latest
2. Download the Windows x64 exe file (iris-windows-amd64.exe)
3. Calculate the SHA256 checksum of the downloaded file
4. Compare with the checksum in chocolateyinstall.ps1

The package maintainer verifies that the download URL points to the official 
releases from the Iris GitHub repository maintained by Arane Aimer (lordaimer).

You can also verify the source code at: https://github.com/lordaimer/iris
